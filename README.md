# Car Care Clinic Redesign

A modern automotive service website redesign focused on improving usability, service discoverability, and the booking experience through a cleaner and more user-centered interface.

## Features
- Simplified navigation
- Improved service organization
- Conversion-focused booking flow
- Clear pricing visibility
- Responsive layouts
- Modern service cards
- Cleaner information hierarchy
- Search feature  and categories

## Tools Used
- Figma

## Design Focus
The redesign focused on improving user flow, reducing confusion, and creating a cleaner and more modern automotive booking experience.
